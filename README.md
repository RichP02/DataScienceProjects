# Data Science Projects
Este repositorio reúne distintos programas y proyectos personales relacionados con **Data Science**, creados con fines de aprendizaje, experimentación y aplicación práctica.

---

## 🏁 Objetivos
- **Aprendizaje**: Reforzar conocimientos en áreas clave de Data Science, incluyendo análisis de datos, estadística, machine learning, visualización y manejo de bases de datos.
- **Demostración**: Mostrar de manera organizada los conocimientos adquiridos a través de proyectos aplicados. Cada carpeta funciona como evidencia del progreso.

---

## 📂 Estructura del repositorio
Cada carpeta corresponde a un proyecto independiente:

- `game_sales/` → Este estudio analiza las ventas de videojuegos mediante un proceso que abarca desde la limpieza de datos hasta la obtención de información clave.
- `predictive-models/` → implementación de algoritmos de machine learning.
- `time-series/` → predicciones y análisis de datos secuenciales.
- `nlp-projects/` → procesamiento de texto y proyectos de NLP.
- `practical-projects/` → aplicaciones completas con datasets reales.

---

## 🛠️ Tecnologías utilizadas
- **Python** (NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn)
- **Jupyter Notebooks** para documentación interactiva
- **Visualización** con librerías gráficas
